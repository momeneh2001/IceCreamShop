## Project Description

**Ice Tales** is an online ice cream e-commerce website built with **Next.js, React, and TailwindCSS**, featuring a **custom API backend** for managing products and users.  
The project aims to provide an interactive and engaging user experience using **animations, maps, sliders, and timers**.

---

## Features

- Display products list and product details  
- Shopping cart and order management  
- User authentication with JWT and cookies  
- Branch maps using **Leaflet**  
- Product sliders with **Swiper**  
- Sales charts and statistics with **Recharts**  
- Discount timers using **react-timer-hook**  
- Alerts and dialogs using **SweetAlert2**  
- Scroll animations with **AOS**  
- Fully responsive layout with TailwindCSS and custom breakpoints

---

## Technologies

- **Frontend:** Next.js, React, TailwindCSS, TypeScript  
- **Backend:** Node.js, MongoDB, Mongoose, JWT, bcryptjs  
- **Libraries:** Axios, Swiper, Leaflet, Recharts, SweetAlert2, react-timer-hook, react-select  
- **Tools:** ESLint, PostCSS, Autoprefixer
